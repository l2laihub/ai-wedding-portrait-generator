Admin Auth Test

Open the browser console and run these commands to test:

// 1. Check if authService is available
authService

// 2. Get current user
authService.getCurrentUser()

// 3. Check if user is admin
authService.isAdmin()

// 4. Refresh user profile to get latest role
await authService.refreshUserProfile()

// 5. Check admin status again
authService.isAdmin()

// 6. Check user role directly
authService.getCurrentUser()?.role

To navigate to admin dashboard: Go to /admin
